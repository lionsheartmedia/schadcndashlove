# Adnova Dashboard — Local Dev Startup

## Start the dev server

```powershell
cd "C:\Users\ryanp\OneDrive\Desktop\LOVE THIS DASH\schadcndashlove"
npx vite --port 8001
```

Then open **http://localhost:8001** in your browser.

## Stop the server

Press `Ctrl + C` in the terminal.
